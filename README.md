# Exam Hall Allotment System

## Requirements
- Windows 7 (32-bit) or newer
- Python 3.8.x (32-bit) or newer

## How to Run
1. Ensure Python is installed.
2. Open the folder containing these files.
3. Double-click `main.py` or run the following command in Command Prompt:
   ```
   python main.py
   ```

## Default Login
- **User ID:** admin
- **Password:** admin123

## Features
1. **Staff Registration:** Add, Update, Delete staff details.
2. **Exam Hall Details:** Manage hall capacity and gender allocation.
3. **Hall Allotment:** Automatically assign staff to halls based on designation rules.
   - HOD/Associate Prof: Assigned 2 times.
   - Assistant Prof: Assigned 3 times.
